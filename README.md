# 🛍️ FashionX - E-commerce Web Application

FashionX is a modern and responsive e-commerce web application built with React.js and Tailwind CSS.  
It provides a smooth online shopping experience with product browsing, category filtering, search functionality, wishlist, cart management, and dark mode support.

The project focuses on creating a clean UI, responsive design, and user-friendly shopping experience.

---

## ✨ Features

### 🏠 Product Browsing
- Responsive product card layout
- View products with image, name, category, and price
- Category based product filtering
- Search products by name

### 🛒 Cart Management
- Add products to cart
- Remove products from cart
- Increase/decrease product quantity
- Dynamic total price calculation
- Cart data persistence using Local Storage

### ❤️ Wishlist
- Add products to wishlist
- Remove products from wishlist
- Add wishlist items directly to cart
- Wishlist persistence using Local Storage

### 🌙 Dark Mode
- Light and dark theme support
- Smooth theme switching
- Tailwind CSS dark mode implementation

### 📱 Responsive UI
- Fully responsive design
- Mobile, tablet and desktop compatible
- Fixed navbar and bottom navigation experience

### 🧭 Navigation
Implemented using React Router:

- Home
- Cart
- Wishlist
- Profile
- Notification
- Contact
- References
- Location
